# 🔬 Alphalytics

**Live Demo → [alphalytics.streamlit.app](https://anmolrastogi21-alphalytics.streamlit.app)**

---

## About

Alphalytics is a real-time stock market dashboard with ML-powered price forecasting. Built from scratch as my first ML project while self-studying Machine Learning and DSA in my 2nd year of CSE.

---

## What it does

- Live stock prices, charts and technical indicators (RSI, SMA)
- Company profile pulled live from Yahoo Finance
- AI-based Buy / Hold / Sell signal
- 7-day price forecast using a Random Forest model (R² 0.9921)
- Stock comparison and multiple timeframes

---

## Built with

Python · scikit-learn · Plotly · Streamlit · yfinance
